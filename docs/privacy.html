Privacy Policy

Last updated: July 22, 2026

Overview

X Reply Sorter is a browser extension that automatically sorts replies on X (formerly Twitter) by your preferred order. Your privacy is important — this extension is designed to collect zero user data.

Data Collection

X Reply Sorter does not collect, store, transmit, or share any personal data. Specifically:

  • No browsing history is collected
  • No user accounts or authentication required
  • No analytics or tracking scripts
  • No data is sent to any server
  • No cookies are set by the extension

Local Storage

The extension stores your sort preference (e.g. "Likes", "Recent", or "Relevant") and enabled/disabled state locally in your browser using Chrome's Storage API. This data never leaves your device and is only used to remember your settings between sessions.

Permissions

The extension requests the following permissions:

  • storage — to save your sort preference locally
  • activeTab — to communicate sort preference changes to the current tab
  • Host permissions for x.com and twitter.com — to run the content script that applies your sort preference on tweet pages

Third-Party Services

The extension does not integrate with any third-party services, APIs, or analytics platforms.

Open Source

The full source code is publicly available on GitHub. You can verify exactly what the extension does.

Changes

If this privacy policy changes, the update will be reflected here with a new "last updated" date.
